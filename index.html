📖 Configurer EmailJS

Suivez ces étapes pour recevoir les fiches clients par email. C'est gratuit jusqu'à 200 emails/mois.

1Créer un compte
Allez sur emailjs.com → Sign Up (gratuit)
2Ajouter votre Gmail
Email Services → Add New Service → Gmail → connectez votre compte Google → notez le Service ID
3Créer un template
Email Templates → Create New → Subject : Nouvelle fiche — {{client_name}}
Body : {{message}}
To email : ink.leblanc@gmail.com → Save → notez le Template ID
4Récupérer votre clé
Account → General → Public Key → copiez la clé
