📖 Configurer EmailJS

Gratuit jusqu'à 200 emails/mois. Suivez ces 4 étapes :

1Créer un compte
Allez sur emailjs.com → Sign Up (gratuit)
2Connecter Gmail
Email Services → Add New Service → Gmail → autorisez toutes les permissions → notez le Service ID
3Créer un template
Email Templates → Create New
• Subject : Nouvelle fiche — {{client_name}}
• To email : ink.leblanc@gmail.com
• Body : {{message}}
→ Save → notez le Template ID
4Récupérer la clé
Account → General → Public Key → copiez

💡 Après configuration, allez dans ⚙️ Réglages pour saisir les identifiants, puis régénérez vos liens d'invitation.
