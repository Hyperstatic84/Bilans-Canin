☁️ Configurer Supabase

Gratuit · ~40 000 cas stockables · Synchronisé sur tous vos appareils.

1Créer un compte
Allez sur supabase.com → Start for free → connectez-vous avec GitHub ou Google
2Créer un projet
New Project → nommez-le "bilans-canin" → choisissez une région proche (ex: West EU) → Create project · Attendez ~1 minute
3Créer la table
Dans votre projet → SQL Editor → New query → Collez ce code et cliquez Run :
create table cases ( id text primary key, data jsonb, ts bigint ); alter table cases disable row level security;
4Récupérer les clés
Project Settings → API
• Copiez l'URL (ex: https://xxxx.supabase.co)
• Copiez l'anon public key (commence par eyJ…)
5Coller dans Réglages
⚙️ Réglages → Supabase → collez l'URL et la clé → Enregistrer et tester

📖 Configurer EmailJS

Gratuit · 200 emails/mois · Vos fiches arrivent directement dans votre boîte mail.

1Créer un compte
Allez sur emailjs.com → Sign Up
2Connecter Gmail
Email Services → Add New Service → Gmail → autorisez toutes les permissions → notez le Service ID
3Créer un template
Email Templates → Create New
• Subject : Nouvelle fiche — {{client_name}}
• To email : ink.leblanc@gmail.com
• Body : {{message}}
→ Save → notez le Template ID
4Récupérer la clé
Account → General → Public Key
