☁️ Configurer Supabase

1Créer un compte
Allez sur supabase.com → Start for free
2Créer un projet
New Project → nommez-le "bilans-canin" → Create project
3Créer la table
SQL Editor → New query → collez et exécutez :
create table cases (id text primary key, data jsonb, ts bigint); alter table cases disable row level security;
4Récupérer les clés
Project Settings → API → copiez l'URL et la clé anon
5Coller dans Réglages
⚙️ Réglages → Supabase → Enregistrer et tester

📖 Configurer EmailJS

1Créer un compte
emailjs.com → Sign Up
2Connecter Gmail
Email Services → Add New Service → Gmail → autorisez → notez le Service ID
3Créer un template
Subject : Nouvelle fiche — {{client_name}}
To : ink.leblanc@gmail.com · Body : {{message}}
4Public Key
Account → General → Public Key
